# 项目依赖和启动
项目分为前端和后端

*vision ---前端*
* 安装依赖
```bash
yarn install
npm install
```
* 启动项目
```bash
yarn serve
npm run server
```
* 项目打包部署
```
yarn build
npm run build
```
*vision-backed ---后端*

* 安装依赖
```bash
npm install
```
* 启动项目
```
npm run dev
node bin/wwww
```
* 如果是开发模式下，建议使用nodemon
```
npm i nodemon -g 
nodemon bin/www
```
